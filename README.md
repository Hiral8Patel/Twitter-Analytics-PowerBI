# Twitter Analytics Dashboard using Power BI

## Overview

This project was developed as part of my Data Analytics Internship at ElevanceSkills Technology Private Limited.

The dashboard analyzes Twitter engagement data using Microsoft Power BI and provides interactive visualizations to understand user engagement and tweet performance.

---

## Tools Used

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- CSV Dataset

---

## Dashboard Features

### Task 1
Tweet Interaction Breakdown by Category

### Task 2
Engagement Rate Comparison

### Task 3
Media Interaction by Day of Week

### Task 4
Replies, Retweets and Likes Comparison

### Task 5
Monthly Engagement Rate Trend

### Task 6
Top 10 Tweets by Engagement
